# Hotel Booking Analytics Dashboard

## Overview

An interactive Streamlit dashboard built using Python, Pandas, Plotly, and Streamlit to analyze hotel booking data and uncover insights related to customer behavior, cancellation patterns, booking trends, and revenue performance.

## Features

* Executive Dashboard
* Data Distribution & Outlier Analysis
* Univariate Analysis
* Bivariate Analysis
* Multivariate Analysis
* Interactive Plotly Visualizations
* Dynamic Filters
* Downloadable Dataset

## Tech Stack

* Python
* Pandas
* NumPy
* Plotly
* Streamlit

## Key Insights

* Transient customers contribute the highest number of cancellations.
* City Hotels experience higher cancellation rates than Resort Hotels.
* ADR varies significantly across room categories.
* Lead Time has a noticeable impact on booking behavior.
* Premium room types generate higher average daily revenue.
